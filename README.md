# 🎮 Trail Dodge — ランダム方向から敵が襲ってくる避けるゲーム

**Trail Dodge** は、  
光のプレイヤーをマウス/タッチで操作して、  
ランダムな方向から飛んでくる敵を避け続けるシンプルな避けゲームです。

生存した時間（秒）がスコアになります。

---

## ✨ 特徴

- 🟡 **光の軌跡（Trail）** を残しながら動くプレイヤー  
- 🎯 **敵は画面の4方向から出現**（上・下・左・右）  
- 🔄 **進行方向もランダム**（中心方向＋ランダムな角度）  
- 🎨 **敵の見た目がランダム**（丸・四角・三角＋ランダム色）  
- 💥 **ぶつかったら即ゲームオーバー**  
- 🧮 **スコアは“生存秒”**

---

## 🕹️ 操作方法

- **マウス**：カーソルに向かってプレイヤーが追従  
- **スマホ/タブレット**：指を動かして移動  
- **Start/Restart** ボタンでゲーム開始

---

1つの HTML ファイルだけ--

## 仕組み

### ● プレイヤー
- 小さな光の円
- 軌跡（Trail）は配列に座標を保存してフェード描画

### ● 敵
- 画面の端（上/右/下/左）からスポーン
- 三角／四角／円の3タイプ
- カラーはランダム
- 中心方向＋ランダム角度で移動する

### ● 衝突
- **円 vs 円** のシンプルな距離チェックで判定  
  （形状は無視して簡略化して高速化）
